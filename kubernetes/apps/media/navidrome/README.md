# Navidrome

Music streaming server with Subsonic API.

- **Namespace:** media
- **Clusters:** Talos II only
- **Access:** Tailscale (`navidrome:4533`)
- **Chart:** app-template v4.6.2
- **Storage:** 2Gi data (SQLite), 50Gi music library
- **Mobile apps:** Symfonium (Android), play:Sub (iOS) via Subsonic API
