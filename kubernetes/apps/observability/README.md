# Observability Namespace

This directory manages the applications and configurations for the `observability` namespace within the Kubernetes cluster.

## Applications

- **uptime-kuma**: A fancy self-hosted monitoring tool.
- **victoria-logs**: Fast, open source and scalable log management system.
- **victoria-logs-collector**: Log collection service for victoria-logs.
- **victoria-metrics**: Fast, cost-effective and scalable monitoring solution and time series database.
